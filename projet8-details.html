Recycle4Pay - Recyclage Rémunérateur

Un système mobile pour récompenser les ménages pour le recyclage des plastiques

Description du projet

Recycle4Pay est une application mobile que j'ai développée avec FlutterFlow. Elle vise à encourager la population à recycler les bouteilles plastiques en leur offrant des gains financiers pour chaque unité rapportée. Chaque bouteille recyclée permet aux ménages de cumuler des points (1 point = 1 FCFA). Chaque semaine, les collecteurs passent dans les quartiers pour récupérer les bouteilles et procèdent directement au paiement via l'application. Les bouteilles sont ensuite revendues aux entreprises partenaires.

Tarification

Paiements aux ménages :
1,5 L = 50 FCFA ; 1 L = 30 FCFA ; 0,5 L = 25 FCFA ; 0,33 L = 20 FCFA ; 10 L = 200 FCFA

Prix de revente aux entreprises :
1,5 L = 65 FCFA ; 1 L = 45 FCFA ; 0,5 L = 40 FCFA ; 0,33 L = 35 FCFA ; 10 L = 250 FCFA

Fonctionnalités principales

  • Application mobile pour les ménages (clients)
  • Application mobile pour les collecteurs
  • Dashboard de gestion pour le suivi des transactions et des collectes
  • Historique des paiements et des points cumulés
  • Suivi des zones desservies et planification des tournées de collecte
  • Statistiques hebdomadaires et mensuelles
